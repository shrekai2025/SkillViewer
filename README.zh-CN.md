# SkillViewer

[English (English Guide)](README.md)

SkillViewer 是一款本地优先的 macOS Agent Skill 浏览器和管理聚合工具。它负责把散落在全系统各个角落的（如 `~/.agents`、`~/.claude`、`~/.codex`、`~/.openclaw` 以及任意你指定的自定义目录）的 agent 技能和 prompt 提示词文件，收录到了一个沉浸式图库化的桌面应用里，带给你类似于使用 Eagle 图库一样的流畅体验。

## 项目架构与技术栈

- **前端框架与渲染**：`Electron + React + TypeScript + Vite`
  使用 Electron 稳定承载本地文件系统，配合极为成熟快速的 React 热刷新体系，能轻易地跑通一条原生应用到 Web Component 的完美打包、签名链路。
- **进程模型边界**：`Electron Main + Preload + Renderer`
  所有的重度扫盘操作（如正则解析 Frontmatter，读取长 Markdown，检查文件重命名禁用）皆留在主进程执行，借助 IPC Preload 层提供优雅安全的 React 交互，避免页面假死卡顿。
- **持久化方案**：`JSON + LocalStorage`
  用户习惯（UI 筛选设定、字号、网格大小、系统浅色/深色外观）悉数存在安全迅速的 `LocalStorage`，多方来源列表配置存储在全局 `userdata` 中。

## 最新核心功能

- **沉浸化拆分布局**：
  双击网格任意一部位立刻进入阅读视口！在全屏编辑器里包含了可收展侧边栏实时 Markdown 预览、强大的无感防抖自动草稿保存机制，以及 `Cmd+S` 主动落盘的系统原生提示支持。
- **安全的真・物理系统级启停逻辑**：
  不搞“UI伪渲染隐藏”，我们对停用的技能做了深度支持（以 `.disabled` 重命名或挂载在停用专属独立文件夹里的能力包），只要在这里关闭，就是真正的系统级全线禁止失效，且它绝不会删你额外的关联文件。
- **全局拦截与状态流监控**：
  即便你企图切回网格、双击其余模块甚至在强行关机重启时，系统依然能在未存储文件时精准捕获并予以弹窗强阻断，守护每一行手写的微小更新。
- **无缝连接的 Agent 技能市场 (Marketplace)**：
  支持直连读取和管理外部 Registry 的技能列表清单文件（manifest），聚合检索一键直达安装，与系统原生环境高度融合。
- **完善的跨端滤网聚合**：
  提供一整套包含自选源漏斗筛选器（系统来源 / 自定义来源 / 仅启用 / 系统级别停用 / 全局状态）的状态聚合器，所有操作一键到底。
- **全面多语言支持 (i18n)**：
  自动跨越系统语言设置，支持用户手动覆璗且应用重启后持久保留，当前支持简体中文、英语和日语三个语言包。
- **Market 来源技能的来源版本检测与一键更新**：
  通过 Marketplace 安装的技能将拖拽完整的版本原孙元数据，SkillViewer 会自动检测是否有可用的新版本，并支持一键就地升级。

## 下载安装与 macOS 安全限制说明

由于本项目是以个人开发者身份免费分发，目前提供的 Release 安装包**没有经过 Apple 开发者昂贵的企业签名**。

当你下载并在 macOS 系统中第一次直接双击打开下载好的 `.app` 时，系统常常会跳出 **“应用已损坏，无法打开。您应该将它移到废纸篓”** 或 **“软件来自未认证开发者”** 等强行阻断提醒。

要绕过这个安全限制以正常运行测试，请在你把应用拖动到 `应用程序 (Applications)` 文件夹后，打开终端 (Terminal) 执行以下解除隔离命令（可能需要输入开机密码）：

```bash
sudo xattr -rd com.apple.quarantine /Applications/Skillviewer.app
```

## 开发与构建指南

在这个根目录下执行以下步骤启动并体验它的极速：

```bash
# 安装所有的配套依赖 Node modules
npm install

# 唤起高速热更新测试开发服务器 
npm run dev

# 若想要打出 DMG 封装安装包
npm run build
npm run dist:mac
```
